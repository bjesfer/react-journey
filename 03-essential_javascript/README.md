# Essential JavaScript for React - Practice Activity

Welcome to the **Essential JavaScript Practice Activity**! This directory contains exercises to help you master all the JavaScript concepts needed for React development.

---

## üéØ What is This Activity?

This activity provides hands-on practice with essential JavaScript concepts that are heavily used in React. You'll work through exercises covering:

1. Variables (const, let, var)
2. Destructuring Objects and Arrays
3. Rest/Spread Operator
4. Template Literals
5. Ternaries Instead of if/else Statements
6. Arrow Functions
7. Short-Circuiting and Logical Operators (&&, ||, ??)
8. Optional Chaining
9. Array Methods (map, filter, reduce, sort, find, etc.)
10. Working with Immutable Arrays
11. Asynchronous JavaScript: Promises
12. Asynchronous JavaScript: Async/Await
13. Modules: Import and Export
14. Default Parameters

---

## üìÅ Files in This Activity

### `start.js`

**What it is:**
- Exercise file with TODO items
- Practice problems for each topic
- Space for your solutions

**How to use:**
1. Open `start.js` in your code editor
2. Work through each TODO item
3. Write your solutions
4. Test your code by running `node start.js`
5. Compare with `final.js` when done

### `final.js`

**What it is:**
- Complete solutions for all exercises
- Best practices and examples
- Reference for learning

**How to use:**
1. Don't look at this until you've attempted exercises!
2. Compare your solutions after completing exercises
3. Learn from different approaches
4. Reference for best practices

## üêõ Common Issues

### Issue: Syntax Errors

**Solution:**
- Check for missing commas, parentheses, brackets
- Verify arrow function syntax
- Check template literal backticks

### Issue: Understanding Concepts

**Solution:**
- Read the documentation guide first
- Break down complex examples
- Practice simpler versions first

### Issue: Immutability

**Solution:**
- Remember: always create new arrays/objects
- Use spread operator
- Use array methods that return new arrays
